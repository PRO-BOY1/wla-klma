# ساحة الكلمات — لعبة تمثيل وتخمين

لعبة ويب عربية للتمثيل والتخمين بين فريقين (🔴 أحمر و🔵 أزرق)، مبنية بـ Next.js 15 وTypeScript وTailwind CSS، وجاهزة للنشر مباشرة على Vercel.

## التشغيل محليًا

```bash
npm install
npm run dev
```

افتح `http://localhost:3000` في المتصفح.

## البناء للإنتاج

```bash
npm run build
npm start
```

## النشر على Vercel

ارفع المشروع إلى GitHub ثم استورده في [vercel.com/new](https://vercel.com/new) — لا حاجة لأي إعدادات إضافية، Vercel يتعرف على Next.js تلقائيًا.

## هيكل المشروع

```
data/cards.json          # كل الكلمات والتصنيفات (تُستخدم كبيانات أولية)
src/app/                 # صفحات Next.js (App Router)
  page.tsx               # شاشة اللعب الرئيسية
  admin/page.tsx          # لوحة إدارة الكلمات
src/components/          # مكونات الواجهة
src/hooks/                # useGameEngine (آلة حالة اللعبة) وuseTimer
src/lib/                  # أنواع، تخزين محلي، أدوات مساعدة، كونفيتي
```

## البيانات والتخزين

- الكلمات والتصنيفات الأولية موجودة في `data/cards.json`، ويمكنك إضافة آلاف الكلمات مباشرة في هذا الملف دون لمس أي كود.
- عند فتح اللعبة أول مرة تُنسخ هذه البيانات إلى `localStorage`، وأي إضافة/تعديل/حذف من لوحة `/admin` يُحفظ هناك تلقائيًا.
- لوحة الإدارة تدعم أيضًا استيراد وتصدير ملف JSON كامل، وزر لاستعادة الكلمات الافتراضية من `data/cards.json`.

## المزايا

- شاشة إعداد كاملة: أسماء الفريقين، عدد الجولات، مدة الجولة، نقاط الفوز الاختيارية، واختيار التصنيفات.
- عرض الكلمة ثم إخفاؤها عند الضغط على "قرأت الكلمة" مع مؤقّت دائري متوهج.
- نافذة نتيجة الدور (فريقي / الفريق الخصم / لم يخمنها أحد) بعد انتهاء الوقت.
- شاشة نهاية احترافية مع Confetti وعرض الفائز والإحصائيات.
- تصميم داكن Glassmorphism بخطوط Cairo وTajawal، متجاوب بالكامل على الجوال والتابلت وسطح المكتب.
